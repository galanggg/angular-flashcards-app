## Prerequisite

`Node v20.x.x`

## Instaling

Run `npm i`

## Start the application

Run `npm run s:flashcards-feature` to start the development server. Happy coding!

## Build for production

Run `npx nx build flashcards-app` to build the application. The build artifacts are stored in the output directory (e.g. `dist/` or `build/`), ready to be deployed.
